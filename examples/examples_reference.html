Examples Reference#

This page will provide a brief overview of the varying examples available in the examples folder on GitHub.

archival/#

This contains the old notebooks and data used to prepare the unveiling of the model to DOE. The results are displayed in presentations section of the documentation.

Explanations#

The following notebooks are aimed at demonstrating and explaining various functionality to users.

how_to.ipynb#

This is a Jupyter Notebook version of the How To section of the documentation that allows users to interact with the walk through.

strategy_demonstration.ipynb#

This is a Jupyter Notebook version of the Strategy Demonstration section of the documentation that allows users to interact with the walk through.

metrics_demonstration.ipynb#

This is a Jupyter Notebook version of the Metrics Demonstration section of the documentation that allows users to interact with the varying maintenance strategy models.

Validation#

dinwoodie_validation.ipynb#

This shows the latest results of the model validation using the modeling parameters from Dinwoodie, et. al, 2015 1.

iea_26_validation.ipynb#

This shows the latest results of the model validation using the modeling parameters from IEA, 2016 2.

timing_benchmarks.ipynb#

This notebook is a simple benchmark for comparing run times between releases from NREL’s FY22 & FY23. That said, there has been a change in computers since running this, and the listed times for v0.7 in the notebooks will not match those in this example.

1

Iain Dinwoodie, Ole-Erik V Endrerud, Matthias Hofmann, Rebecca Martin, and Iver Bakken Sperstad. Reference cases for verification of operation and maintenance simulation models for offshore wind farms. Wind Engineering, 39(1):1–14, 2015.

2

Gavin Smart, Aaron Smith, Ethan Warner, Iver Bakken Sperstad, Bob Prinsen, and Roberto Lacal-Arantegui. Iea wind task 26: offshore wind farm baseline documentation. Technical Report, National Renewable Energy Lab.(NREL), Golden, CO (United States), 2016.
